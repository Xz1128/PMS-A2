Troubleshooting

Form won't submit

Ensure all required fields (marked with *) are filled in correctly. Numeric fields must contain valid numbers only.

Cannot find item to edit/delete

The item name search is case-insensitive but must match exactly. Try viewing all items first to confirm the exact name.

Duplicate ID error

Each item must have a unique Item ID. Check the existing items table to find an unused ID number.

Page appears blank

Try refreshing the browser. If the issue persists, open the browser developer console (F12) to check for JavaScript errors.
